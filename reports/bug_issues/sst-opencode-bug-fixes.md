## 条目

1) #4568 — opentui 输入分词报 TypeError  
   - 现象：输入内容不是字符串时调用 `split` 触发异常，导致界面崩溃。  
   - 解决：在处理输入前确保内容为字符串并做空值防护（引用提交 `5413b16b...`）。  
   - 链接：https://github.com/sst/opencode/issues/4568

2) #4523 — 输入 @ 提示时无法中途引用文件  
   - 现象：在句子中间输入空格后再输入 @，文件引用提示无法弹出。  
   - 解决：修复了提及解析的偏移/边界判断，允许空格后继续触发文件引用（修复 PR 4121）。  
   - 链接：https://github.com/sst/opencode/issues/4523

3) #4063 — 升级后提示缺少 API key（Zen 模型）  
   - 现象：调用付费模型时提示缺少密钥，实际凭据已存在。  
   - 解决：发现工作目录下有额外的本地配置文件覆盖了全局认证；删除/调整本地配置后恢复正常。  
   - 链接：https://github.com/sst/opencode/issues/4063
